Member RPC API¶

Member RPCs are exposed under /gov and require authentication with a valid member identity.

POST /ack¶

Example request:

POST /ack HTTP/1.1
Host: example.com
Content-Type: application/json

{
    "state_digest": [
        1
    ]
}
Status Codes

  • 200 OK –

    Default response description

    Example response:

    HTTP/1.1 200 OK
Content-Type: application/json

true
